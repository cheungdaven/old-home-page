Journals




  • An Ontology & Linked Data Driven Semantic Integration Method of Library Information Resources and Its Evaluation
    with Shiyan Ou, Shan Hu.
    Library and Information Service, 2014, 58(2) PDF, Abstract

    This paper seeks to investigate the application of semantic web technologies in information resource integration at libraries. Firstly, the authors proposed an ontology- and linked- data-driven semantic integration framework to realize (1) the integration of document resources of different types and in different formats; (2) semantic integration of resources in different data collections; as well as (3) linking and integration of library resources with external resources, so as to form an open and scalable resource integration system. Then, they built a demo integration system based on this framework. Finally, the authors compared this demo system with CALIS OPAC and CNKI which use traditional integration modes.
